
# 📘 دليل إدارة موقع TechTouch (المرجع الشامل)

أهلاً بك في الدليل التشغيلي لموقع **TechTouch**. هذا الموقع مبني بنظام **Static Website** (ملفات ثابتة HTML/CSS/JS)، مما يجعله سريعاً جداً وآمناً.

---

## 📂 دليل ملفات المشروع (أين أجد...؟)

إليك خريطة تفصيلية للتحكم في كل جزء من أجزاء الموقع:

### 1️⃣ التحكم في الإحصائيات والإعلانات (Google Analytics & AdSense)
*   **الملف:** `scripts/generate.js` (الأهم)
    *   **الوظيفة:** هذا هو المحرك الذي يبني صفحات الموقع. ستجد في بدايته متغيرات ثابتة:
        *   `GA_ID`: ضع هنا كود تتبع **Google Analytics**.
        *   `AD_CLIENT_ID`: ضع هنا معرف **Google AdSense**.
        *   `ONESIGNAL_APP_ID`: ضع هنا كود إشعارات **OneSignal**.
    *   **تنبيه:** بعد تعديل هذا الملف، يجب تشغيل أمر البناء `npm run build` لتطبيق التغييرات على جميع الصفحات.

*   **الملف:** `index.html` (والصفحات الأخرى)
    *   يحتوي على الأكواد المحقونة بالفعل. إذا كنت تعدل يدوياً دون السكربت، عدل الكود هنا في قسم `<head>`.

### 2️⃣ المحتوى والبيانات (قاعدة البيانات)
*   **المجلد:** `content/posts/`
    *   يحتوي على ملفات `.json` (مثل `asus-gx10.json`). كل ملف يمثل **مقالاً واحداً**. عدل النص هنا ليتغير في الموقع.
*   **المجلد:** `content/data/`
    *   `about.json`: يتحكم في **معلومات الموقع بالكامل** (اسم الموقع، صورتك، البايو، روابط السوشيال ميديا، نصوص البوت، والبحث).
    *   `channels.json`: يتحكم في القائمة الموجودة في صفحة "الأدوات > دليل القنوات".
*   **الملف:** `assets/js/phones-data.js`
    *   يحتوي على قاعدة بيانات الهواتف لأداة "مقارنة الهواتف". أضف هواتف جديدة هنا.

### 3️⃣ التصميم والمظهر (Frontend)
*   **الملف:** `assets/css/style.css`
    *   يحتوي على التنسيقات الخاصة (مثل شكل شريط الأخبار المتحرك، وتنسيق النصوص).
*   **الملف:** `assets/js/app.js`
    *   يتحكم في: الوضع الليلي/النهاري، التنقل بين التبويبات، وشريط الأخبار.

### 4️⃣ لوحة التحكم (Admin Panel)
*   **المجلد:** `admin/`
    *   `admin.js`: الملف المسؤول عن تشغيل لوحة التحكم والاتصال بـ GitHub.

---

## 🚀 لوحة التحكم (CMS) - المميزات

يمكنك إدارة الموقع بسهولة عبر لوحة التحكم الموجودة في المجلد `/admin`.

### 1. إضافة فيديوهات وروابط في المقالات
عند كتابة مقال جديد، ستجد أزراراً مساعدة فوق صندوق المحتوى:
*   **زر يوتيوب (Youtube):** الصق رابط الفيديو ليظهر كمشغل داخل المقال.
*   **زر رابط (Link):** لإضافة أزرار التحميل الملونة.

### 2. تعديل معلومات "من نحن" والبوت
*   اذهب إلى تبويب **"الملف الشخصي والمظهر"** في لوحة التحكم.
*   يمكنك تعديل نصوص "بوت الطلبات" و "طرق البحث" وحفظها مباشرة.

---

## 📍 أوامر التشغيل (للمطورين)

بعد إجراء التغييرات من لوحة التحكم أو يدوياً في ملفات `content/`، يجب إعادة بناء الموقع ليتم تحديث صفحات HTML.

1.  افتح الطرفية (Terminal).
2.  نفذ الأمر:
    ```bash
    npm run build
    ```
    *أو*
    ```bash
    node scripts/generate.js
    ```
3.  سيقوم هذا السكربت بقراءة ملفات JSON وتحديث جميع ملفات HTML (index, about, articles, sitemap) تلقائياً.
